How Airside ⟶ LogTen Importer works

Use this guide to export the correct files, convert them locally, and import the finished CSV into LogTen.

1. Export your Airside roster / flight log

  1. Open Airside.
  2. Go to Statistics.
  3. Press Export Flight Log.
  4. Select the date range you want to export.
  5. Enter the email address where Airside should send the file.
  6. Press Export.
  7. Download the CSV attachment from the email and upload it here as Flight CSV.

2. Export your LogTen People Export

This step is optional, but recommended. It lets the importer match PIC employee IDs to LogTen names and detect your own name for SIC using This is Me.

  1. Open LogTen.
  2. Go to People.
  3. Export the People Export file.
  4. Upload that TXT or CSV file here as LogTen People export.

3. What the importer changes

  • Splits Airside flight values such as SK5896-20240527-CPH-MMX into flight number, departure, and destination.
  • Optionally formats aircraft registrations, for example SERSX to SE-RSX.
  • Optionally changes E95 to E190.
  • Optionally replaces PIC employee IDs with matching names from the LogTen People Export.
  • Optionally fills SIC from the LogTen person marked This is Me, or from the manual SIC fallback field.
  • Shows missing PIC IDs so you can type names manually before downloading.
  • Optionally scans AirLabs historical data for approved Employee IDs and SAS SK flights through your Wasmer backend and adds Scheduled Dep and Scheduled Arr columns.

4. Fix missing PIC IDs in LogTen

If the importer shows missing PIC IDs, use Download unknown PIC IDs to get a small CSV with the employee numbers that were not found in your LogTen People Export.

  1. Open LogTen and go to People.
  2. Add or edit the missing pilot.
  3. Put the employee number into that person's ID field.
  4. Make sure the name is written exactly how you want it imported as PIC/P1 Crew.
  5. Export a fresh People Export from LogTen.
  6. Upload the new People Export here and run the conversion again.

You can also type a temporary manual PIC fix in the preview tab, but updating LogTen People is better because future imports will match automatically.

5. Import into LogTen

After downloading the finished CSV, use the LogTen import mapping panel on this page. It shows exactly which CSV columns to map to which LogTen fields and categories.

6. Privacy

Normal conversion happens inside this browser tab. If scheduled-time lookup is enabled and your Employee ID is approved, only the Employee ID, flight number, date, and route are sent to your own Wasmer backend so it can check cache and call AirLabs without exposing the API key. The API key and allowed Employee IDs are managed on the private admin page.

Tip: Check the live preview and the Ready to import card before opening LogTen's importer.
