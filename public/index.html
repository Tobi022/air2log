Welcome to Airside ⟶ LogTen Importer

Convert your Airside roster export into a LogTen-ready CSV with optional crew matching and scheduled times.

What this tool is made for

Airside ⟶ LogTen Importer takes the Flight Log CSV from Airside and prepares it for LogTen import. It keeps the workflow private and reduces the manual cleanup normally needed before importing flights.

  • Splits Airside flight strings into flight number, departure airport, and destination airport.
  • Formats aircraft registrations, for example SERSX to SE-RSX.
  • Converts aircraft type E95 to E190.
  • Matches PIC employee IDs to pilot names from your LogTen People Export.
  • Fills SIC from the person marked This is Me, or from your manual fallback.
  • Optionally adds Scheduled Dep and Scheduled Arr from AirLabs through your private Wasmer backend.

Privacy

Normal CSV conversion happens locally in this browser tab. Your Flight CSV and LogTen People Export are not uploaded anywhere.

Only when Scheduled times is enabled does the app send limited flight lookup data to your own private Wasmer backend: flight number, flight date, departure airport, arrival airport, and the Employee ID needed for access control. The AirLabs API key stays in the admin page/database and is never exposed in the public HTML.

Step 1 — Export from Airside

  1. Open Airside.
  2. Go to Statistics.
  3. Press Export Flight Log.
  4. Select the date range you want to export.
  5. Enter the email address where Airside should send the file.
  6. Press Export.
  7. Download the CSV attachment from the email and upload it here as Flight CSV.

Step 2 — Export LogTen People Export

This is optional, but recommended. It enables automatic PIC name matching and SIC auto-fill.

  1. Open LogTen.
  2. Go to People.
  3. Export the People Export file.
  4. Upload that TXT or CSV file here as LogTen People export.

Step 3 — Review before download

  • Check the Ready to import card in the Summary panel.
  • Review missing PIC IDs and type temporary manual fixes if needed.
  • Review scheduled-time results and the Scheduled Time Note column.
  • Use the live preview to confirm the first converted rows look right.

Fix missing PIC IDs in LogTen

If the importer finds unknown PIC IDs, press Download unknown PIC IDs. This creates a small CSV with the employee numbers that were not found in your LogTen People Export.

  1. Open LogTen and go to People.
  2. Add or edit each missing pilot.
  3. Put the employee number into that person’s ID field.
  4. Make sure the name is written exactly how you want it imported as PIC/P1 Crew.
  5. Export a fresh People Export from LogTen.
  6. Upload the new People Export here and run the conversion again.

You can also type a temporary manual PIC fix in the preview tab, but updating LogTen People is better because future imports will match automatically.

Scheduled times

Scheduled-time lookup is only shown when the uploaded Flight CSV contains an approved Employee ID. It only checks SAS SK flight numbers and converts values such as SK0602 to SK602 before lookup.

Scheduled times are only accepted when they are within 60 minutes of the actual CSV block times. If a match is too far away, the scheduled-time columns stay empty and the reason is written into Scheduled Time Note.

Step 4 — Import into LogTen

Download the converted CSV and import it into LogTen. Use the LogTen import mapping panel in the Summary area for the exact fields and categories.

The top help button shows this overview. The Full mapping guide button in the Summary panel still opens the detailed LogTen field mapping.
